# Bandwidth Tester

This script measures your internet connection speed.

It performs a download and upload speed test using the Speedtest service.

## Features

- Measure download speed
- Measure upload speed
- Automatic server selection

## Requirements

Python 3

Install required library:

pip install speedtest-cli

## Usage

Run the script:

python bandwidth_tester.py

The script will automatically test your internet speed.

## Example Output

Bandwidth Tester
-----------------------------

Download Speed: 94.21 Mbps
Upload Speed  : 39.87 Mbps
