# Port Scanner Script

This script scans common network ports on a target host.

It helps identify which ports are open and accessible.

## Features

- Scan common ports
- Fast port checking
- Useful for basic network analysis

## Requirements

Python 3 installed.

Check version:

python --version

## Usage

Run the script:

python port_scanner.py

Enter a target IP or domain when prompted.

Example:

google.com

## Example Output

Scanning target: google.com

Port 80 is OPEN  
Port 443 is OPEN

Scan finished.
