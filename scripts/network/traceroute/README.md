# Traceroute Script

This script performs a traceroute to a target host.

Traceroute shows the path that network packets take from your system to the destination server.

## Features

- Displays routing path
- Useful for diagnosing network issues
- Simple command line usage

## Requirements

Python 3

This script uses the built-in system traceroute command.

## Usage

Run the script:

python traceroute.py

Enter a host or IP address when prompted.

Example:

google.com
