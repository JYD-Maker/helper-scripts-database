# DNS Resolver

This script resolves a domain name to its IP address using DNS.

It can be used to test DNS resolution and verify domain connectivity.

## Features

- Resolve domain names
- Display IP address
- Simple command line usage

## Requirements

Python 3

This script uses Python's built-in `socket` module.

## Usage

Run the script:

python dns_resolver.py

Enter a domain name.

Example:

google.com
