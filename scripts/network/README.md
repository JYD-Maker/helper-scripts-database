# Network Scripts

This section contains networking related helper scripts.

The tools in this directory help with network diagnostics, connectivity checks and network analysis.

## Available Scripts

| Script | Description |
|------|------|
| DNS Lookup | Resolve a domain name to its IP address |
| Port Scanner | Scan common ports on a target host |
| Ping Monitor | Continuously monitor host availability |
| IP Info | Retrieve geographic and network information for an IP |
| HTTP Status Checker | Check if a website is reachable and return HTTP status |
| Traceroute | Display the route packets take to a destination |
| Subnet Calculator | Calculate subnet details from CIDR notation |
| Local Network Scanner | Discover active devices in a local network |
| DNS Resolver | Resolve domain names using DNS |
| Bandwidth Tester | Measure internet download and upload speed |

## Usage

Navigate to the desired script directory and run the script.

Example:

python dns_lookup.py

## Requirements

Most scripts require:

- Python 3
- Internet connection (for some tools)

Some scripts may require additional libraries listed in their individual README files.
