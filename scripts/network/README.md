# Network Scripts

This section contains helper scripts related to networking tasks.

These scripts are designed to help developers, system administrators and IT professionals troubleshoot and analyze network connections.

## Available Scripts

### DNS Lookup
Resolve a domain name into its IP address.

Location:
dns-lookup/

Features:
- Fast domain resolution
- Simple command line usage
- Useful for troubleshooting DNS problems

More network scripts will be added over time.
