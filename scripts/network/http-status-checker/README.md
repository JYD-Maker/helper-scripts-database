# HTTP Status Checker

This script checks whether a website is reachable and returns the HTTP response status.

It is useful for testing website availability and debugging connection issues.

## Features

- Check HTTP response codes
- Detect if a website is reachable
- Simple command line interface
- Clear output for quick diagnostics

## Requirements

Python 3 installed.

Install the required library:

pip install requests

## Usage

Run the script:

python http_status_checker.py

Enter a website URL when prompted.

Example:

https://example.com

## Example Output

HTTP Status Checker
-----------------------------

URL: https://example.com
Status Code: 200
Status: Website is reachable.
