# DNS Lookup Script

This script performs a simple DNS lookup and resolves a domain name to its IP address.

It can be useful for network troubleshooting, testing DNS resolution, or learning how DNS works in practice.

## Features

- Resolve domain names to IP addresses
- Simple command line interface
- Lightweight and fast

## Requirements

Python 3 must be installed.

Check your Python version:

python --version

## Usage

Run the script:

python dns_lookup.py

Then enter a domain name:

example.com

## Example Output

Domain: example.com  
IP Address: 93.184.216.34

## Use Cases

- Checking DNS resolution
- Network troubleshooting
- Learning basic networking concepts
