# Disk Usage

This script displays disk usage statistics of the system.

It shows total space, used space and available space.

## Features

- Shows disk capacity
- Displays used storage
- Displays available storage
- Simple command line output

## Requirements

Python 3

No additional libraries required.

## Usage

Run the script:

python disk_usage.py

## Example Output

Disk Usage
--------------------------
Path: /
Total Space: 512 GB
Used Space: 210 GB
Free Space: 302 GB
