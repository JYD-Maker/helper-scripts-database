# File Organizer

This script organizes files inside a directory based on their file type.

Files are automatically moved into folders such as:

- Images
- Documents
- Videos
- Archives
- Scripts

## Features

- Automatic file sorting
- Creates folders automatically
- Supports common file types

## Requirements

Python 3

No additional libraries required.

## Usage

Run the script:

python file_organizer.py

Enter the directory you want to organize.

Example:

C:\Users\User\Downloads
