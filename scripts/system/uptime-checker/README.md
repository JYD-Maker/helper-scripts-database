# Uptime Checker

This script displays how long the system has been running since the last reboot.

It retrieves the system boot time and calculates the uptime.

## Features

- Displays system start time
- Shows total uptime
- Useful for system monitoring

## Requirements

Python 3

Install required library:

pip install psutil

## Usage

Run the script:

python uptime_checker.py

## Example Output

System Uptime
--------------------------------
System started at: 2026-03-12 07:45:11
Uptime: 1 day, 2:14:31
