# Temp Cleaner

This script deletes files from the system temporary directory.

Temporary files are often created by applications and may accumulate over time.

Cleaning them can free up disk space.

## Features

- Deletes temporary files
- Removes temporary folders
- Displays number of deleted items

## Requirements

Python 3

No additional libraries required.

## Usage

Run the script:

python temp_cleaner.py

The script will clean the system temporary folder automatically.

## Example Output

Temporary File Cleaner
------------------------------

Temp Directory: C:\Users\User\AppData\Local\Temp

Deleted items: 128
