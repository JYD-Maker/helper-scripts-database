# File Search

This script searches for files inside a directory based on a keyword.

It scans the specified directory and all subdirectories.

## Features

- Search files by name
- Recursive directory scanning
- Displays full file paths

## Requirements

Python 3

No additional libraries required.

## Usage

Run the script:

python file_search.py

Enter:

1. Directory path
2. Keyword to search for

Example:

Directory:
C:\Logs

Keyword:
error

## Example Output

File Search
