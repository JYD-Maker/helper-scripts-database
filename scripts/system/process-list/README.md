# Process List

This script displays all running processes on the system.

It shows the process ID, name and CPU usage.

## Features

- Lists active processes
- Displays process IDs
- Shows CPU usage
- Useful for system monitoring

## Requirements

Python 3

Install required library:

pip install psutil

## Usage

Run the script:

python process_list.py
