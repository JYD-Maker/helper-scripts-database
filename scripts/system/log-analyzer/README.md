# Log Analyzer

This script analyzes log files and searches for specific keywords.

It helps identify important entries such as errors or warnings.

## Features

- Search log files for keywords
- Displays line numbers
- Useful for troubleshooting

## Requirements

Python 3

No additional libraries required.

## Usage

Run the script:

python log_analyzer.py

Enter:

1. Path to the log file
2. Keyword to search

Example:

Log file:
C:\logs\system.log

Keyword:
error

## Example Output

Log Analyzer
--------------------------------

Found 2 matching entries:

[Line 45] ERROR: Connection failed
[Line 132] WARNING: Disk usage high
