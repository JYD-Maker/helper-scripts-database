# Service Checker

This script lists Windows services and displays their current status.

It helps administrators check whether services are running or stopped.

## Features

- Displays Windows services
- Shows service status
- Useful for system diagnostics

## Requirements

Python 3

Install required library:

pip install psutil

## Usage

Run the script:

python service_checker.py

## Example Output

-------------------------------------------
