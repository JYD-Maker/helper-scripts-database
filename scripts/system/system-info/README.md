# System Scripts

This directory contains helper scripts related to system administration and system diagnostics.

The scripts in this category provide useful information about the current system and help monitor system resources.

## Available Scripts

| Script | Description |
|------|------|
| system_info.py | Displays system information such as OS, CPU and RAM |

## Requirements

Python 3

Some scripts may require additional libraries.

For this script install:

pip install psutil

## Usage

Run the script:

python system_info.py

The script will display basic system information automatically.

## Example Output

System Information
--------------------------------
Hostname: MY-PC
Operating System: Windows 11
Processor: Intel(R) Core(TM)
CPU Cores (Physical): 8
CPU Threads (Logical): 16
Total RAM: 32 GB
Available RAM: 21 GB
