# CPU Monitor

This script monitors CPU usage in real time.

It displays the current processor utilization at regular intervals.

## Features

- Real-time CPU monitoring
- Simple command line output
- Lightweight system monitoring tool

## Requirements

Python 3

Install required library:

pip install psutil

## Usage

Run the script:

python cpu_monitor.py

The script will display CPU usage continuously.

Stop the program with:

CTRL + C

------------------------------

CPU Usage: 21%
