# Firewall Simulator

This script simulates a simple firewall by allowing or blocking IP addresses.

It checks whether a given IP is in a blocked list and decides if access should be allowed or denied.

It is useful for learning how firewall rules work and understanding basic network security concepts.

## Features

- Allow or block IP addresses
- Simple rule-based system
- Fast and lightweight
- Command line interface

## Requirements

Python 3

## Usage

Run the script:

python firewall_simulator.py

Enter an IP address when prompted.

Example:

192.168.1.10

## Example Output

Firewall Simulator  
Enter IP address: 192.168.1.10  

Access denied for 192.168.1.10 ❌  

## Use Cases

- Learning firewall basics
- Understanding network access control
- Simulating security rules
- Practice for network security concepts
