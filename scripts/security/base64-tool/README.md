# Base64 Tool

This script encodes and decodes text using Base64 encoding.

It is useful for learning encoding techniques and working with encoded data.

## Features

- Encode text to Base64
- Decode Base64 to original text
- Simple command line interface
- Error handling for invalid input

## Requirements

Python 3

## Usage

Run the script:

python base64_tool.py

Select:

1 = Encode  
2 = Decode  

## Example Output

Base64 Tool  
Select option: 1  

Enter text: hello  

Encoded: aGVsbG8=
 
## Use Cases

- Encoding data
- Decoding Base64 strings
- Learning encoding concepts
- Working with APIs and tokens
