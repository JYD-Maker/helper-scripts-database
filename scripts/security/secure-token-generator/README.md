# Secure Token Generator

This script generates secure random tokens using Python's built-in secrets module.

It is suitable for generating session tokens, API keys, and other security-related values.

It is useful for learning secure random generation and understanding token-based authentication.

## Features

- Generate secure random tokens
- Uses cryptographically strong randomness
- Configurable token length
- Simple command line interface

## Requirements

Python 3

## Usage

Run the script:

python secure_token_generator.py

Enter the desired token length when prompted.

Example:

16

## Example Output

Secure Token Generator  
Enter token length: 16  

Secure Token: 9f8d7c6a5b4e3d2c1a...

## Use Cases

- Generating API keys
- Creating session tokens
- Security testing
- Learning secure token generation
