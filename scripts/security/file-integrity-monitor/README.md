# File Integrity Monitor

This script monitors files for changes using SHA256 hashing.

## Features

- Create baseline of file hashes
- Detect modified files
- Detect deleted files
- Simple directory monitoring

## Requirements

Python 3

No additional libraries required.

## Usage

Run the script:

python file_integrity_monitor.py

Select:

1 = Create baseline  
2 = Check integrity  

## Example Output

File Integrity Monitor  
Select option: 1  

Enter directory path: C:\Test  

Baseline created ✅
