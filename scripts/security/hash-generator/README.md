# Hash Generator

This script generates cryptographic hashes from input text.

Supported algorithms include:

- MD5
- SHA1
- SHA256

## Features

- Multiple hashing algorithms
- Simple input-based usage
- Useful for integrity checks

## Requirements

Python 3

No additional libraries required.

## Usage

Run the script:

python hash_generator.py

Enter:

1. Text
2. Algorithm (md5, sha1, sha256)

Enter algorithm: sha256
