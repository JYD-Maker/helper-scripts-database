# File Permission Checker

This script checks the permissions of a file.

It shows whether a file is readable, writable, or executable.

It is useful for understanding file access rights and basic system security.

## Features

- Check file read permission
- Check file write permission
- Check file execute permission
- Simple command line interface

## Requirements

Python 3

## Usage

Run the script:

python file_permission_checker.py

Enter a file path when prompted.

Example:

C:\test.txt

## Example Output

File Permission Checker  
File: C:\test.txt  

Readable ✅  
Writable ✅  
Executable ❌  

## Use Cases

- Checking file access rights
- Learning file permissions
- Basic system security checks
