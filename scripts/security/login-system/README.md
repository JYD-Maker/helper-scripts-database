# Login System

This script provides a simple login system with user registration and authentication.

It allows users to create accounts and securely store passwords using SHA256 hashing.

It is useful for learning authentication basics, password security, and how login systems work in practice.

## Features

- User registration with username and password
- Password hashing using SHA256
- User data stored in a JSON file
- Login verification
- Simple command line interface

## Requirements

Python 3

## Usage

Run the script:

python login_system.py

Select an option:

1 = Register a new user  
2 = Login  

Enter your username and password when prompted.
